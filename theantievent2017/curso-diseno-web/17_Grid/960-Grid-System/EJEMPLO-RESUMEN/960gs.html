960 Grid System

Incluir las CSS: text, reset y 960 css's en el html

Podemos utilizar 12, 16 o 24 columnas para crear nuestros layouts, cada una de las columnas tiene un margen izquierdo y derecho de 10px, para crear 20px de separación entre columnas. Sabiendo esto, decimos que si utilizo 12 columnas, cada columna será de 60px cada una y sí utilizo 16 columnas, el ancho de cada una será de 40px.

Pero,¿Cómo 960 grid sabe sí utilizo 12 o 16 columnas? Simple, tiene una clase .container_yy, donde "yy" es el numero de columnas que voy a utilizar.

Cada una de las filas lleva un número, que será el identificador; es decir, el ancho de la columna corresponde al número de columnas que abarca. Y se forma así: grid_xx donde "xx" es el numero de columnas.

Finalizar cada fila con un div clear: <div class="clear"></div>

Class alpha y class omega: sirven cuando estamos anidando unos grid dentro de otros, para ajustar los márgenes de los distintos elementos anidados (recordar las clases grid_xx tienen un margen de 10 píxeles a la izquierda y la derecha; se sumarían los márgenes de elemento padre e hijos).

Class prefix_xx: sirve para anclar una grid_xx dejando un espacio vacío a la izquierda; y class suffix_xx: para colocar un espacio vacío a la derecha

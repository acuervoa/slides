Unsemantic Grid System

1- mirar el head y copiar todos los css y js (da soporte a IE 7 y 8)
2- contenedor principal: class="grid-container" (ancho max 1222px aprox y min 320px). A los 767 px todas las columnas se ponen unas debajo de otras (para móviles)
3- cada columna: class="grid-XX" donde XX es el % de ancho, y son múltiplos de 5 (5,10,15,20,.....80,90,95,100). ej: class="grid-80"
4- se pueden poner espacios vacíos (márgenes) a izquierda: class="grid-70 prefix-30" o a derecha: class="grid-60 suffix-40"
5- se pueden anidar columnas: al div padre poner la clase "grid-parent"
